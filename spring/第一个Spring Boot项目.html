# 创建Spring Boot项目

创建一个满足如下要求的Spring Boot项目。

  • 依赖Web、H2、JPA、Thymeleaf

  • Kotlin语言

  • Gradle构建

  • Spring版本为1.5.10.RELEASE

  • Group为tech.daking

  • Artifact为readinglist

  • Name为Reading List

  • 描述为Reading List Demo

  • 包名为tech.daking.readinglist

通常使用Spring Initializer在线来创建。

也可以使用Spring Boot CLI中的spring init命令来创建。

spring init -d=web,data-jpa,h2,thymeleaf --build=gradle -l=kotlin -b="1.5.10.RELEASE" -p=jar -g="tech.daking" -a=readinglist -n="Reading List" --description="Reading List Demo" --package-name="tech.daking.readinglist" readlinglist.zip

# Spring Boot项目结构

Spring Boot项目遵循传统Maven或Gradle项目的布局。

  • 主要代码位于src/main/kotlin目录中。

  • 资源位于src/main/resources目录中。

  • 测试代码在src/test/kotlin目录中。

  • 测试资源在src/test/resources目录中。

readinglist
├── build.gradle // Gradle构建说明文件
└── src
    ├── main
    │   ├── kotlin
    │   │   └── tech
    │   │       └── daking
    │   │           └── readinglist
    │   │               └── ReadingListApplication.kt // 应用程序的启动引导类，也是主要的Spring配置类
    │   └── resources
    │       ├── application.properties // 用于配置应用程序和Spring Boot的属性
    │       ├── static
    │       └── templates
    └── test
        └── kotlin
            └── tech
                └── daking
                    └── readinglist
                        └── ReadingListApplicationTests.kt // 基本的集成测试类

# 启动引导Spring

/* ReadingListApplication.kt */
package tech.daking.readinglist

import org.springframework.boot.SpringApplication
import org.springframework.boot.autoconfigure.SpringBootApplication

@SpringBootApplication // 开启组件扫描和自动配置
class ReadingListApplication

fun main(args: Array<String>) {
    SpringApplication.run(ReadingListApplication::class.java, *args) // 启动引导应用程序
}

@SpringBootApplication是将以下几个注解组合在一起。

  • @Configuration，表明该类使用Spring基于Java的配置。

  • @ComponentScan，启用组件扫描。

  • @EnableAutoConfiguration，开启Spring Boot的自动配置。

# 项目构建脚本

/* build.gradle */
buildscript {
    ext {
        kotlinVersion = '1.2.20'
        springBootVersion = '1.5.10.RELEASE'
    }
    repositories {
        mavenCentral()
    }
    dependencies {
        classpath("org.springframework.boot:spring-boot-gradle-plugin:${springBootVersion}")
        classpath("org.jetbrains.kotlin:kotlin-gradle-plugin:${kotlinVersion}")
        classpath("org.jetbrains.kotlin:kotlin-allopen:${kotlinVersion}")
    }
}

apply plugin: 'kotlin'
apply plugin: 'kotlin-spring'
apply plugin: 'eclipse'
apply plugin: 'org.springframework.boot'

group = 'tech.daking'
version = '0.0.1-SNAPSHOT'
sourceCompatibility = 1.8
compileKotlin {
    kotlinOptions {
        freeCompilerArgs = ["-Xjsr305=strict"]
        jvmTarget = "1.8"
    }
}
compileTestKotlin {
    kotlinOptions {
        freeCompilerArgs = ["-Xjsr305=strict"]
        jvmTarget = "1.8"
    }
}

repositories {
    mavenCentral()
}


dependencies {
    compile('org.springframework.boot:spring-boot-starter-data-jpa')
    compile('org.springframework.boot:spring-boot-starter-thymeleaf')
    compile('org.springframework.boot:spring-boot-starter-web')
    compile("org.jetbrains.kotlin:kotlin-stdlib-jdk8:${kotlinVersion}")
    compile("org.jetbrains.kotlin:kotlin-reflect:${kotlinVersion}")
    runtime('com.h2database:h2')
    testCompile('org.springframework.boot:spring-boot-starter-test')
}
