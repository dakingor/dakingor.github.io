# ViewModel原理

ViewModel主要用来在Activity/Fragment因设备配置改变导致重建时能保存一些数据。

ViewModelStore用来保存ViewModel实例。

  • Fragment级别的ViewModel存储在Fragment实例中的ViewModelStore类型成员变量里。

  • Activity级别的ViewModel存储在FragmentActivity实例中的ViewModelStore类型成员变量里。

FragmentActivity会保存和恢复ViewModelStore实例。

  1. FragmentActivity利用onRetainNonConfigurationInstance()来保存其ViewModelStore成员变量和所拥有的各Fragment的数据（包括该Fragment的ViewModelStore成员变量）。

  2. FragmentActivity在onCreate(Bundle savedInstanceState)中调用getLastNonConfigurationInstance()获取上面所保存的信息，即可恢复ViewModelStore成员变量和各Fragment的ViewModelStore成员变量。

# ViewModelStore

ViewModelStore的内部是利用HashMap来存储ViewModel实例。

/* ViewModelStore.java */
public class ViewModelStore {

    private final HashMap<String, ViewModel> mMap = new HashMap<>();

    final void put(String key, ViewModel viewModel) {
        ViewModel oldViewModel = mMap.put(key, viewModel);
        if (oldViewModel != null) {
            oldViewModel.onCleared();
        }
    }

    final ViewModel get(String key) {
        return mMap.get(key);
    }

    public final void clear() {
        for (ViewModel vm : mMap.values()) {
            vm.onCleared();
        }
        mMap.clear();
    }
}

# ViewModelProvider

ViewModelProvider包含两个重要的成员变量：ViewModelProvider.Factory实例和ViewModelStore实例。这两个成员变量是在构造函数中传入。

/* ViewModelProvider.java */
private final Factory mFactory;
private final ViewModelStore mViewModelStore;
    
public ViewModelProvider(@NonNull ViewModelStore store, @NonNull Factory factory) {
    mFactory = factory;
    mViewModelStore = store;
}

通过ViewModelProvider#get()获取ViewModel实例的流程是：

  1. 先从ViewModelStore成员变量中获取，若存在并符合要求则直接返回，否则进行下一步。

  2. 利用Factory成员变量创建对应的ViewModel实例，并把该实例存入ViewModelStore成员变量中，最后返回该实例。

/* ViewModelProvider.java */
@NonNull
@MainThread
public <T extends ViewModel> T get(@NonNull String key, @NonNull Class<T> modelClass) {
    ViewModel viewModel = mViewModelStore.get(key);

    if (modelClass.isInstance(viewModel)) {
        return (T) viewModel;
    }

    viewModel = mFactory.create(modelClass);
    mViewModelStore.put(key, viewModel);
    return (T) viewModel;
}

# ViewModelProvider.Factory

在ViewModelProvider类中，ViewModelProvider.Factory有两个默认实现：NewInstanceFactory和AndroidViewModelFactory。

NewInstanceFactory在create()中实例化出ViewModel对象。

public static class NewInstanceFactory implements Factory {

    @NonNull
    @Override
    public <T extends ViewModel> T create(@NonNull Class<T> modelClass) {
        try {
            return modelClass.newInstance();
        }
        // ...
    }
}

AndroidViewModelFactory持有Application成员变量，并实现单例模式。AndroidViewModelFactory在create()中实例化出AndroidViewModel或ViewModel对象。

public static class AndroidViewModelFactory extends ViewModelProvider.NewInstanceFactory {

    private static AndroidViewModelFactory sInstance;

    @NonNull
    public static AndroidViewModelFactory getInstance(@NonNull Application application) {
        if (sInstance == null) {
            sInstance = new AndroidViewModelFactory(application);
        }
        return sInstance;
    }

    private Application mApplication;

    public AndroidViewModelFactory(@NonNull Application application) {
        mApplication = application;
    }

    @NonNull
    @Override
    public <T extends ViewModel> T create(@NonNull Class<T> modelClass) {
        // 若Class为AndroidViewModel类型或其子类型，利用Application成员变量进行实例化，否则交给父类来实例化。
        if (AndroidViewModel.class.isAssignableFrom(modelClass)) {
            try {
                return modelClass.getConstructor(Application.class).newInstance(mApplication);
            }
            // ...
        }
        return super.create(modelClass);
    }
}

# ViewModelProviders

ViewModelProviders可看作是ViewModelProvider的工厂类或工具类。

ViewModelProviders利用FragmentActivity/Fragment和ViewModelProvider.Factory来创建ViewModelProvider。

/* ViewModelProviders.java */
@NonNull
@MainThread
public static ViewModelProvider of(@NonNull Fragment fragment, @Nullable Factory factory) {
    // 获取Application实例
    Application application = checkApplication(checkActivity(fragment));
    // 若factory为空，则使用默认工厂
    if (factory == null) {
        factory = ViewModelProvider.AndroidViewModelFactory.getInstance(application);
    }
    // 利用Fragment实例中的ViewModelStore成员变量和factory来创建ViewModelProvider
    return new ViewModelProvider(fragment.getViewModelStore(), factory);
}

@NonNull
@MainThread
public static ViewModelProvider of(@NonNull FragmentActivity activity, @Nullable Factory factory) {
    // 获取Application实例
    Application application = checkApplication(activity);
    // 若factory为空，则使用默认工厂
    if (factory == null) {
        factory = ViewModelProvider.AndroidViewModelFactory.getInstance(application);
    }
    // 利用FragmentActivity实例中的ViewModelStore成员变量和factory来创建ViewModelProvider
    return new ViewModelProvider(activity.getViewModelStore(), factory);
}
