# Annotation Processor

# Annotation Processor介绍

Annotation Processor（注解处理器）是javac的一个工具，用来在编译时扫描和处理Annotation（注解）。

你可以自定义一个新的Annotation Processor并注册它来使用。

javac会启动一个独立的JVM来运行Annotation Processor。

# 自定义Annotation Processor

继承javax.annotation.processing.AbstractProcessor类，并按需实现其方法。

public class MyProcessor extends AbstractProcessor {

    @Override
    public synchronized void init(ProcessingEnvironment env){ }

    @Override
    public boolean process(Set<? extends TypeElement> annoations, RoundEnvironment env) { }

    @Override
    public Set<String> getSupportedAnnotationTypes() { }

    @Override
    public SourceVersion getSupportedSourceVersion() { }

}

init方法: 会被注解处理工具调用，并输入ProcessingEnviroment参数。而ProcessingEnviroment提供很多有用的工具类，如Elements, Types和Filer等。

process方法: 相当于每个处理器的主函数。要在这里编写扫描、评估和处理注解的代码，以及生成Java文件。参数RoundEnviroment，可以查询出包含特定注解的被注解元素。

getSupportedAnnotationTypes方法: 它的返回值是一个字符串的集合，包含本处理器想要处理的注解类型的合法全称。

getSupportedSourceVersion方法: 用来指定要使用的Java版本。通常返回SourceVersion.latestSupported()。

# 注册Annotation Processor

Google提供了com.google.auto.service:auto-service库帮助我们方便地注册自定义的Annotation Processor。

首先，导入依赖包。

dependencies {
    implementation 'com.google.auto.service:auto-service:1.0-rc4'
}
1
2
3

然后在自定义Annotation Processor的实现类上添加@AutoService(Processor.class)注解声明。

@AutoService(Processor.class)
public class MyProcessor extends AbstractProcessor {
    ...
}

# android-apt

# android-apt介绍

android-apt是Android Studio中一款用来辅助处理编译时注解的Gradle插件。

android-apt允许设置一个仅编译时的注解处理器作为依赖，这个处理器不会被包含在最终输出的apk或库中。

# android-apt使用

在build.gradle中应用android-apt插件。

buildscript {
    repositories {
        mavenCentral()
    }
    dependencies {
        classpath 'com.neenbedankt.gradle.plugins:android-apt:1.8'
    }
}

apply plugin: 'com.neenbedankt.android-apt'

# Android Gradle Plugin的annotationProcessor

Android Gradle Plugin v2.2添加了名为annotationProcessor的功能来代替第三方的android-apt插件。

可以直接使用annotationProcessor代替apt来依赖一个仅编译时的注解处理器，而无需进行额外的配置。

dependencies {
    compile 'com.google.dagger:dagger:2.0'
    annotationProcessor 'com.google.dagger:dagger-compiler:2.0'
}

# kotlin-kapt

# kotlin-kapt介绍

kapt为Kotlin annotation processing tool的缩写，译为Kotlin注解处理工具。

kotlin-kapt是Android Studio中一款用来辅助处理编译时注解的Kotlin Gradle插件。

kapt会生成Kotlin代码，并将生成的Kotlin源文件写入processingEnv.options["kapt.kotlin.generated"]所指定的目录，这些文件会与主源代码一起编译。

# kotlin-kapt使用

在build.gradle中应用kotlin-kapt插件。

buildscript {
    repositories {
        mavenCentral()
        google()
    }
    dependencies {
        classpath "org.jetbrains.kotlin:kotlin-gradle-plugin:$kotlin_version"
    }
}

apply plugin: 'kotlin-kapt'
