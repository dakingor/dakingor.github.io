# Rx系列

# RxJava

RxJava是Reactive Extensions在JVM上的实现，使用可观察序列来实现异步和基于事件的程序。

RxJava分为RxJava1和RxJava2，因此与它相关的其他Rx库也分成两个大版本。

# RxAndroid

RxAndroid是RxJava针对Android平台的特定API。

RxAndroid是RxJava体系的一部分，不是RxJava的替代，而是RxJava在Android平台上的补充。

# RxKotlin

RxKotlin是基于RxJava的扩展库，以Kotlin风格提供大量扩展方法。

# RxLifecycle

RxLifecycle通过绑定生命周期的方式解决在Android平台上使用RxJava的内存泄露问题。

# RxPermissions

RxPermissions结合RxJava处理Android 6.0起的运行时权限。

# 依赖注入

# ButterKnife

ButterKnife是Android视图的字段和方法绑定，它使用注释处理生成模板代码。

# Kotlin Android Extensions

Kotlin Android Extensions是Kotlin团队开发的插件，增强Android开发体验。

目前利用Kotlin Android Extensions来进行视图绑定，避免引入ButterKnife等第三方类库。

# Daager2

Daager2是应用于Android和Java的依赖注入框架。

# Anko

Anko是一个Kotlin库，它使Android应用开发更加快捷。

Anko的类库体系庞大，主要包含Commons、Layouts、SQLite和Coroutines等几个子库。常使用Commons子库即可。

# 架构

# ARouter

ARouter是一个Android路由中间件，帮助app导航到activity和service。

# Tinker

Tinker是微信开源的Android热修复解决方案。

# 网络层

# OkHttp

OkHttp是一个可用于Android和Java应用程序的HTTP和HTTP2库。

okhttp-utils是对OkHttp的封装工具类库，已停止维护。

# Retrofit

Retrofit是Square公司开发的适用于Android和Java的类型安全的HTTP客户端框架。

Retrofit官网

# 视图库

# AndroidAutoLayout

AndroidAutoLayout是一套Android屏幕适配方案，已停止维护。

# MZBannerView

MZBannerView是仿魅族应用市场中的Banner的图片轮播控件。

MZBannerView支持多种模式切换：普通ViewPager、普通Banner和仿魅族风格的Banner。

# FlycoTabLayout

FlycoTabLayout是一个TabLayout库。

# MagicIndicator

MagicIndicator是一个强大、可定制、易扩展的ViewPager指示器框架。

MagicIndicator是ViewPagerIndicator、TabLayout、PagerSlidingTabStrip等的替代品。

MagicIndicator支持角标，更支持在非ViewPager场景下使用。

# Android-AlertView

Android-AlertView是模仿iOS的AlertViewController的弹窗控件，支持Alert和ActionSheet模式。

# sweet-alert-dialog

sweet-alert-dialog是一个优雅简洁的Alert Dialog库。

# TimePickerDialog

TimePickerDialog是一个滚轮样式的时间选择器。

# PatternLockView

PatternLockView是一个易用的、符合MD风格的图形锁界面库。

# PasscodeView

PasscodeView是一个数字密码界面库。

# BottomNavigation

BottomNavigation是一个MD风格的底部导航控件。

# Banner

Banner是一个图片轮播控件。

# TriangleLabelView

TriangleLabelView是一个三角形标签控件，借助它可在某个视图的角落位置设置三角形标签。

# BadgeView

BadgeView是一个支持自由定制外观、可拖拽消除的MD风格的徽章控件。

# NumberButton

NumberButton，购物车的商品数量、增加和减少控制按钮的控件。

# Coverflow

Coverflow是一个封面浏览控件。

# MultiStateView

MultiStateView是一个多状态控件，根据当前状态显示不同的内容。

# FlowLayout

FlowLayout是一个流式布局控件，支持单选、多选等。

# Virtualview-Android

Virtualview-Android是阿里开源的一款可利用自定义XML构建UI的轻量级库。

# BGAQRCode-Android

BGAQRCode-Android，扫描二维码、扫描条形码、相册获取图片后识别、生成带 Logo 二维码、支持微博/微信/QQ的二维码扫描样式。

# BadgeView

BadgeView，支持自由定制外观、拖拽消除的MaterialDesign风格Android BadgeView。

# Colorful

Colorful，支持运行时换肤。

# 刷新视图

# BGARefreshLayout

BGARefreshLayout可对各种控件实现多种下拉刷新效果、上拉加载更多以及配置自定义头部广告位。

# SmartRefreshLayout

SmartRefreshLayout支持各种View和多层嵌套视图结构的下拉刷新。

# RecyclerView

# BaseRecyclerViewAdapterHelper

BaseRecyclerViewAdapterHelper，简称BRVAH，是一个强大的RecyclerAdapter框架。

BRVAH简书教程

# vlayout

vlayout是一个为RecyclerView而扩展的强大LayoutManager。

vlayout为RecyclerView提供了一组布局，以解决在同一个RecyclerView上同时需要grid、list和其他布局等复杂场景。

# FlexibleDivider

FlexibleDivider

# 图像库

# Glide

Glide是一个Android图片加载和缓存库。

# CircleImageView

CircleImageView是圆形ImageView。

# PhotoView

PhotoView是一款支持通过多种的触摸手势来进行缩放的ImageView。

# StickerView

StickerView是一个贴纸库。

# Android GPUImage

Android GPUImage是一个基于OpenGL的滤镜库，它是参考iOS GPUImage实现的。

# TakePhoto

TakePhoto是一款用于在Android设备上获取照片（拍照或从相册、文件夹中选择）、裁剪图片、压缩图片的开源工具库。

# 视频库

# JiaoZiVideoPlayer

JiaoZiVideoPlayer，原名为JieCaoVideoPlayer。

# 动画库

# Lottie

Lottie是Airbnb开源的动画库。

将Adobe After Effects上设计的动画通过Bodymovin插件导出为.json文件。Lottie可引用此json文件进行动画实现。

Lottie支持Android、iOS、React Native等平台。

# Keyframes

Keyframes是Facebook开源的动画库。

Keyframes将Adobe After Effects上设计的动画转换为自身能解析的数据格式，并根据这份数据来实现动画。

Keyframes支持Android、iOS平台。

# 数据库

# DBFlow

DBFlow，A blazing fast, powerful, and very simple ORM android database library.

# Realm

Realm是基于C++编写的一个不同于SQLite的数据库引擎。

相对于SQLite以及其ORMlibraries，Realm更轻量级、更快且具有很多现代数据库的特性，如支持JSON、流式api、数据变更通知以及加密支持等。

# GreenDAO

GreenDAO是基于SQLite数据库的轻量级、高效的ORM解决方案。

# ORMLite

ORMLite是基于SQLite数据库的ORM解决方案。

# objectbox

objectbox-java是一个超快的移动端数据库。

# 交互

# JsBridge

JsBridge是一个受微信的WebView的js桥梁启发而诞生的Android与js交互的类库。

# 性能

# AndroidGodEye

AndroidGodEye一款可以在PC浏览器中实时监控App内存泄漏的库。

# 文档

# MuPDF

MuPDF is a lightweight PDF, XPS, and E-book viewer.

# 通信

# NanoHttpd

NanoHttpd是一个小型、轻量级、可嵌入的HTTP服务框架。

# AndroidAsync

AndroidAsync是一个支持异步Socket、HTTP客户端/服务端、WebSocket的Android库。它基于NIO实现。

# swiftp

swiftp在Android设备上建立FTP服务。

# FastBle

FastBle，Android BLE的快速开发框架。

# 图表

# hellocharts

hellocharts-android，最低兼容API8，提供多种类型的图表组件，支持缩放、滚动和动画。

# androidplot

androidplot，最低兼容Android 1.6，用于创建动态或静态的图表。

# WilliamChart

WilliamChart

# MPAndroidChart

MPAndroidChart，支持多种类型多种效果的强大的图表库。

# 功能

# AndPermission

AndPermission，适配国内各大ROM的动态权限申请框架。

# AndResGuard

AndResGuard是微信团队开发的Android资源混淆库。

# AndroidMultiChannelBuildTool

AndroidMultiChannelBuildTool是python脚本编写的Android多渠道打包工具，只支持v1签名。

# packer-ng-plugin

packer-ng-plugin，Android多渠道打包工具。

# fat-aar

fat-aar支持Android Gradle Plugin 3的fat-aar插件。
