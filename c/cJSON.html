# 基本概念

cJSON是轻量级的JSON解析器。

整个库只有cJSON.h和cJSON.c两个文件，代码量只有几百行。使用时只需把这两个文件加入到自己的工程中。

# cJSON结构

typedef struct cJSON {   
    struct cJSON *next, *prev;   // 用于数组对象中查找前或后一个cJSON结构   
    struct cJSON *child;         // 数组和对象中指向子数组对象或值  
    
    int type;                    // 元素类型，数组、对象等  
    // 存储的数据，三选一 
    char* valuestring;           // 1.获取存储的字符串   
    int valueint;                // 2.获取存储的数字   
    double valuedouble;          // 3.获取存储的cJSON_Number   
    
    char *string;                // item名称  
} cJSON;

# 释放cJSON对象

函数原型为：

void cJSON_Delete(cJSON* c);
1

示例如下：

cJSON_Delete(root);
1

注意，此方法会释放cJSON*对象的child、valuestring、string部分，并递归到其next继续执行释放，直至尾。
